## deno-api-generator

This is a CLI API generator using TypeScript and Deno.
Using this you will be able to generate controllers, services and models for 
a given resource.

**Work in progress**

---------

### How to use

Generate controller 

```
generate -c controller_name
```

Generate service 

```
generate -s service_name
```

Generate model 

```
generate -m model_name
```
